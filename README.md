# 我的愿望打卡本

**个人生活记录工具**

---

## 简介

我的愿望打卡本是一款帮助个人记录每日小目标、养成好习惯的微信小程序。通过设定目标、每日打卡、积分积累的方式，让生活更有规划。

---

## 核心功能

### 1. 记录目标
- 快捷预设小目标（今晚做个好梦、本周状态满分等）
- 自定义目标内容
- 选择目标难度和实现周期

### 2. 每日打卡
- 记录今日行动
- 积累积分
- 连续打卡获得加成奖励

### 3. 我的记录
- 查看所有进行中的目标
- 追踪每个目标的完成进度
- 可视化进度条

### 4. 积分系统
- 完成任务获得积分
- 连续打卡积分加成
- 达成目标解锁成就

---

## 使用场景

- 设定短期小目标（1-3天）
- 规划中期计划（7-14天）
- 追踪长期习惯（30-90天）
- 每日自我提升打卡
