## 二、重新规划界面流程

### 2.1 新流程设计

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   步骤1         │ →  │   步骤2         │ →  │   步骤3         │ →  │   步骤4         │
│ 选择行动类型     │    │ 选择具体行动     │    │ 补充描述        │    │ 评估得分        │
│ (大图标卡片)     │    │ (网格小图标)     │    │ (可选输入)      │    │ (滑动评分)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 2.2 步骤1：选择行动类型

**设计要点：**
- 2列网格布局
- 每个卡片包含：大图标(56rpx) + 类型名称 + 简要描述
- 选中状态：渐变背景 + 白色文字 + 上浮效果
- 未选中状态：白色背景 + 灰色边框

**交互效果：**
- 点击时轻微缩放(0.96)
- 选中卡片阴影增强
- 图标有轻微放大动画

### 2.3 步骤2：选择具体行动 ⭐新增

**设计要点：**
- 3列网格布局
- 每个选项包含：小图标(44rpx) + 行动名称 + 描述
- 点击后选中/取消选中（支持多选）
- 已选行动显示在顶部标签栏
- 自定义输入可折叠

**交互效果：**
- 选中时边框变为主题色
- 图标轻微放大
- 顶部标签可删除

### 2.4 步骤3：补充描述 ⭐改进

**设计要点：**
- 已选行动类型和具体行动显示在顶部汇总卡片
- 输入框带有图标前缀(📝)
- 快捷标签一键添加
- 字数统计实时显示

**视觉改进：**
- 输入框增加左侧图标
- 边框聚焦时发光效果
- 圆角加大到24rpx

### 2.5 步骤4：评估得分

**设计要点：**
- 问题卡片化展示
- 选项使用带emoji的标签式按钮
- 实时分数预览卡片（渐变色）
- 进度条可视化

## 四、样式规范

### 4.1 颜色系统

**主色调：**
- 主色：`#FF6B9D` (粉红)
- 辅色：`#A855F7` (紫色)
- 渐变：`linear-gradient(135deg, #FF6B9D, #A855F7)`

**中性色：**
- 背景：`#FFF5F8` (极浅粉)
- 卡片白：`rgba(255, 255, 255, 0.95)`
- 文字主色：`#333333`
- 文字次色：`#666666`
- 文字辅助：`#888888`
- 边框：`rgba(255, 107, 157, 0.2)`

### 4.2 尺寸规范

**间距：**
- 页面边距：30rpx
- 卡片间距：24rpx
- 元素间距：16rpx
- 内边距：24rpx-32rpx

**字体：**
- 标题：36-40rpx, bold
- 正文：28-30rpx, normal
- 辅助：24-26rpx, normal
- 小字：20-22rpx, normal

**圆角：**
- 大卡片：32rpx
- 中卡片：24rpx
- 小卡片：16-20rpx
- 按钮：20-24rpx

### 4.3 阴影效果

```css
/* 轻微阴影 */
box-shadow: 0 4rpx 12rpx rgba(0, 0, 0, 0.05);

/* 卡片阴影 */
box-shadow: 0 8rpx 32rpx rgba(255, 107, 157, 0.1);

/* 选中阴影 */
box-shadow: 0 12rpx 40rpx rgba(255, 107, 157, 0.3);
```

## 六、设计说明

### 设计理念

本次优化遵循以下设计原则：

1. **直观引导** - 通过图标和预设选项降低用户思考成本
2. **视觉层次** - 清晰的信息层级，重要内容突出显示
3. **情感化设计** - 粉紫色调传递温暖、积极的情感
4. **一致性** - 统一的视觉语言和交互模式
5. **反馈及时** - 每次操作都有明确的视觉反馈

### 用户体验提升点

| 优化项 | 原问题 | 改进效果 |
|-------|-------|---------|
| 具体行动选择 | 用户不知道写什么 | 42个预设选项一键选择 |
| 图标设计 | 单一缺乏层次 | 大小图标分级展示 |
| 输入框 | 不明显 | 图标前缀+聚焦发光效果 |
| 日历功能 | 缺失 | 完整历史记录查看+统计 |
| 多选支持 | 只能单选 | 支持多个行动同时记录 |
| 快捷输入 | 无 | 快捷标签一键添加 |

### 技术亮点

- **CSS变量**：使用 `--card-color` 动态设置卡片主题色
- **Grid布局**：使用 CSS Grid 实现响应式网格
- **平滑动画**：所有交互都有过渡动画
- **组件化思维**：卡片、标签等可复用组件
